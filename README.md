## 🎯 What ZARii AI Does

ZARii AI is an intelligent agricultural assistant that helps Pakistani farmers diagnose crop diseases and get treatment recommendations — all through voice or text in their local language.

**Speak in Urdu, Punjabi, or English — get expert agricultural advice instantly.**

---

## ✨ Key Features

### For Farmers (User App)

| Feature | What It Does |
|---------|--------------|
| **📸 Crop Disease Detection** | Snap a photo of your crop leaf, get instant AI diagnosis with 95%+ accuracy |
| **💊 Smart Treatments** | Get personalized treatment plans with product recommendations available in Pakistan |
| **🎙️ Voice Assistant** | Talk to ZARii in your own voice — ask questions, get answers in speech |
| **🌍 Weather-Aware** | Every diagnosis comes with local weather context that affects treatment |
| **🌐 Multilingual** | Full support for English, Urdu (اردو), and Punjabi (پنجابی) |
| **📱 Simple to Use** | No registration required — just open and start using |

### For Admins (Admin Panel)

| Feature | What It Does |
|---------|--------------|
| **📊 Dashboard** | See real-time stats: users, scans, revenue |
| **👨‍🌾 Farmer Management** | View and manage all registered farmers |
| **🩺 Diagnosis History** | Browse all disease scans and outcomes |
| **📦 Product Catalog** | Manage pesticides, fertilizers, seeds inventory |
| **🤝 Sponsors** | Feature sponsored products in recommendations |
| **🔑 AI Configuration** | Manage AI providers for vision and voice |
| **👥 Team** | Manage admin team with role-based access |
| **💰 Revenue** | Track earnings and financial metrics |
| **⚠️ Outbreak Alerts** | Monitor disease outbreaks by region |
| **💬 WhatsApp** | Configure WhatsApp bot integration |

## 🚀 How to Use

### As a Farmer

1. **Open the app** — No download needed
2. **Choose input** — Upload a photo OR use voice
3. **Get results** — Receive diagnosis + treatment
4. **Follow advice** — Apply recommended products

### As an Admin

1. Go to `your-site.com/#admin`
2. Login with admin credentials
3. Access dashboard and all management features
